# Farm Labour Attendance Tracker

A web application to track farm worker attendance, manage payroll, and generate monthly summaries.

## Features
- Worker attendance tracking
- Monthly summary reports
- Payroll calculation
- User-friendly interface

## Getting Started

### Prerequisites
- Node.js
- pnpm (or npm/yarn)

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/rakshith123-cyber/farm-attendence.git
   ```
2. Install dependencies:
   ```sh
   pnpm install
   ```
3. Start the development server:
   ```sh
   pnpm run dev
   ```

## Folder Structure
- `src/` - Frontend source code
- `backend/` - Backend logic
- `build-template/` - Deployment scripts and configs
